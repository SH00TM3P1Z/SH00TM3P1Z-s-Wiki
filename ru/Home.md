# Enhanced Dogtags - Вики настройки жетонов

Документация по всем настройкам жетонов в моде Enhanced Dogtags от SH00TM3P1Z (SMPZ).

> **Синхронизация в реальном времени:** Все настройки жетонов (DogtagSettings, AIDogtagSettings, DogtagTradeSettings, JSON по категориям и т.д.) **применяются на сервере в реальном времени при сохранении файлов**, поэтому перезапуск сервера не требуется для изменения конфигурации.

## Пути к файлам конфигурации

Все файлы конфигурации хранятся относительно папки профиля сервера.

| Путь | Описание |
|------|------|
| `$profile:EnhancedDogtag\Config\` | Папка конфигурации |
| `$profile:EnhancedDogtag\Config\DogtagSettings.json` | Базовые настройки жетона (лут, Death Wish и т.д.) |
| `$profile:EnhancedDogtag\Config\AIDogtagSettings.json` | Настройки ИИ жетона |
| `$profile:EnhancedDogtag\Config\DogtagTradeSettings.json` | Основные настройки обмена жетонов |
| `$profile:EnhancedDogtag\Config\Category\` | JSON файлы предметов обмена по категориям |
| `$profile:EnhancedDogtag\Data\DogtagLifetime.json` | Данные жизненного цикла жетона (автогенерация, ручное редактирование не рекомендуется) |

> **Примечание:** `$profile:` — путь к профилю сервера DayZ. (напр., `DayZServer\Profiles\`)
