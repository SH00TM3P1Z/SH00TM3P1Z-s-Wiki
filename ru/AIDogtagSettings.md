# Настройки ИИ жетона (AIDogtagSettings)

**Путь к файлу:** `$profile:EnhancedDogtag\Config\AIDogtagSettings.json`

Файл конфигурации, управляющий тем, могут ли ИИ носить жетоны и можно ли лутить жетоны с трупов ИИ.

---

## Настройки

| Опция | Тип | По умолчанию | Описание |
|------|------|--------|------|
| `Version` | int | 2 | Версия конфига (изменять не рекомендуется) |
| `AIDogtagEnabled` | bool | true | Включить ИИ жетоны (могут ли ИИ носить жетоны) |
| `AIDogtagLootingEnabled` | bool | true | Разрешить лут ИИ жетонов (можно ли лутить жетоны с трупов ИИ) |

### Подробное поведение

**AIDogtagEnabled:** Если false, ИИ не носят жетоны. Жетоны не выпадают с трупов ИИ.

**AIDogtagLootingEnabled:** Если false, игроки не могут лутить жетоны с трупов ИИ, даже если они есть. Лут блокируется даже при AIDogtagEnabled = true.

Обе настройки должны быть true для получения ИИ жетонов и их использования в обмене.

---

## Пример конфигурации

```json
{
  "Version": 2,
  "AIDogtagEnabled": true,
  "AIDogtagLootingEnabled": true
}
```

---

## Связь с DogtagSettings

| 파일 | 용도 |
|------|------|
| **DogtagSettings.json** | Базовое поведение жетона игрока (лут, Death Wish и т.д.) |
| **AIDogtagSettings.json** | Поведение ИИ жетона (ношение ИИ/лут) |

ИИ жетоны можно использовать в системе обмена. Используются вместе с настройками `AIDogtag` в [Условия обмена (TradeConditions)](TradeConditions).
