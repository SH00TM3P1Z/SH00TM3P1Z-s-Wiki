# Enhanced Dogtags - Dogtag Settings Wiki

Documentation for all dogtag-related settings in the Enhanced Dogtags mod by SH00TM3P1Z (SMPZ).

> **Real-time Sync:** All dogtag settings (DogtagSettings, AIDogtagSettings, DogtagTradeSettings, category JSON files, etc.) are **applied to the server in real-time when files are saved**, so no server restart is required for configuration changes.

## Config File Paths

All config files are stored relative to the server profile folder.

| Path | Description |
|------|------|
| `$profile:EnhancedDogtag\Config\` | Config folder |
| `$profile:EnhancedDogtag\Config\DogtagSettings.json` | Dogtag base settings (looting, Death Wish, etc.) |
| `$profile:EnhancedDogtag\Config\AIDogtagSettings.json` | AI dogtag settings |
| `$profile:EnhancedDogtag\Config\DogtagTradeSettings.json` | Dogtag trade main settings |
| `$profile:EnhancedDogtag\Config\Category\` | Category-specific trade item JSON files |
| `$profile:EnhancedDogtag\Data\DogtagLifetime.json` | Dogtag lifetime data (auto-generated, manual edit not recommended) |

> **Note:** `$profile:` refers to the DayZ server profile path. (e.g., `DayZServer\Profiles\`)
