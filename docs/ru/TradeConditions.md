# Условия обмена (TradeConditions)

Секция `TradeConditions` в `DogtagTradeSettings.json` определяет **глобальные условия обмена**.

## Структура

```json
{
  "TradeConditions": {
    "Enabled": 1,
    "PlayerDogtag": {
      "Enabled": true,
      "NeedDeathWishDogtag": 0,
      "RequireSelfKill": 1,
      "RequireAny": 0,
      "AllowSuicideDogtag": 0
    },
    "AIDogtag": {
      "Enabled": true,
      "RequireSelfKill": 1,
      "RequireAny": 0
    }
  }
}
```

---

## Enabled (глобально)

| Значение | Описание |
|----|------|
| 1 | Глобальные условия обмена включены. Применяются ко всем предметам (кроме предметов с индивидуальными условиями) |
| 0 | Глобальные условия обмена выключены. Обмен возможен только когда JSON предмета имеет `ItemTradeConditions.Enabled=1` |

> **Доступность обмена:** Обмен разрешён, когда либо глобальный `Enabled`, либо `ItemTradeConditions.Enabled` предмета = 1. Если оба 0, обмен блокируется и появляется уведомление "Trade Fail".

---

## PlayerDogtag (жетоны игроков)

| Опция | Тип | Описание |
|------|------|------|
| `Enabled` | bool | Разрешить обмен жетонами игроков (1=разрешить, 0=запретить) |
| `NeedDeathWishDogtag` | bool | Только жетоны Death Wish (1=только Death Wish, 0=включая обычные жетоны) |
| `RequireSelfKill` | bool | Только жетоны от собственных убийств (1=только свои, 0=разрешены любые) |
| `RequireAny` | bool | (Не используется) |
| `AllowSuicideDogtag` | bool | Разрешить обмен жетонами при самоубийстве (1=разрешить, 0=исключить) |

### Подробное поведение PlayerDogtag

**NeedDeathWishDogtag:** Если 1, только жетоны Death Wish можно использовать для обмена. Обычные жетоны нельзя. Если 0, можно использовать и обычные, и Death Wish жетоны.

**RequireSelfKill:** Если 1, только жетоны от собственных убийств можно использовать. Жетоны от убитых другими игроками нельзя. Если 0, можно использовать любой жетон независимо от того, кто убил.

**AllowSuicideDogtag:** Если 1, жетоны при самоубийстве можно использовать для обмена. Если 0, жетоны при самоубийстве исключаются из обмена.

---

## AIDogtag (жетоны ИИ)

| Опция | Тип | Описание |
|------|------|------|
| `Enabled` | bool | Разрешить обмен жетонами ИИ (1=разрешить, 0=запретить) |
| `RequireSelfKill` | bool | Только жетоны ИИ от собственных убийств (1=только свои, 0=разрешены любые) |
| `RequireAny` | bool | (Не используется) |

### Подробное поведение AIDogtag

**RequireSelfKill:** Аналогично PlayerDogtag. Если 1, только жетоны ИИ от собственных убийств. Если 0, можно использовать любой жетон ИИ.

---

## Приоритет условий

| Глобальный Enabled | Item ItemTradeConditions.Enabled | Применяемые условия |
|--------------|-----------------------------------|----------------|
| 1 | 0 | Глобальные TradeConditions (предметы без индивидуальных условий используют глобальные) |
| 0 | 1 | Item ItemTradeConditions (только этот предмет использует другие условия) |
| 1 | 1 | Item ItemTradeConditions (индивидуальные условия переопределяют глобальные) |
| 0 | 0 | Обмен заблокирован (TRADE_FAIL) |

Если у предмета есть `ItemTradeConditions` с `Enabled` 1, этот предмет игнорирует глобальные настройки и использует только свои условия PlayerDogtag и AIDogtag.
